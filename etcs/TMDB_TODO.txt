Image loading in blocked regions (revisit later)
- Problem: `CORS_PROXIES` helps only for TMDb API JSON (server-side requests to `api.themoviedb.org`), not for browser `<img src="https://image.tmdb.org/...">`.
- Option A (fastest): Keep direct `https://image.tmdb.org/t/p/...` URLs (CDN).
- Option B (works when images blocked): Add a Django image proxy endpoint (e.g. `/tmdb-image/<size>/<path>`) that fetches images server-side and can fall back through `CORS_PROXIES`.
- Option C (best-of-both): Hybrid client behavior — try direct TMDb image URL first, and on error swap to the proxy URL.
